## Configura tu proyecto

* Configurar `php/conexion.php` con los detalles de su base de datos
```php
$db_host = "localhost";
$db_user = "username";
$db_pass = "password";
$db_name = "database";
```

* Puedes importar la base de datos o ejecutar los comandos del archivo `sql/proy_taller.sql`
