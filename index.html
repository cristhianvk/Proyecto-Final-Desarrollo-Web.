¿Qué aprenderás en este curso?

Todo lo necesario y obligatorio de HTML, CSS y JavaScript, realizar la maquetacion de sitios web y asi asentar las bases del Desarrollo Web y estar preparado para profundizar en el campo.

HTML 5

Lenguaje de marcado HTML5 utilizado en el desarrollo web para crear el contenido y la estructura de una página web.

CSS 3

CSS es un lenguaje de diseño gráfico que permite definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

JavaScript

JavaScript es el lenguaje de programación encargado de dotar de mayor interactividad y dinamismo a las páginas web.
